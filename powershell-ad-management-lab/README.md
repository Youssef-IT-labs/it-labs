# PowerShell AD Management Lab

## Overview
This project demonstrates how to use PowerShell to automate Active Directory user management tasks in a Windows Server environment.

The goal is to simulate practical sysadmin work such as creating users, assigning them to the correct organizational unit, and adding them to the appropriate security groups.

## Scope
This lab focuses on user provisioning in an existing Active Directory environment.

## Goals
- Create Active Directory users with PowerShell
- Avoid hardcoded passwords in scripts
- Place users in the correct OU
- Add users to the correct AD groups
- Build a reusable and practical admin script
